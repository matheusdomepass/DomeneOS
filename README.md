# DomeneOS

Sistema de gerenciamento de clientes e ordens de serviço desenvolvido para estudo e portfólio utilizando **ASP.NET Core MVC**.

---

##  Sobre o projeto

O **DomeneOS** é um sistema desenvolvido com o objetivo de simular a gestão de uma assistência técnica, permitindo o gerenciamento de clientes e ordens de serviço.

O projeto foi criado para aplicar conceitos de **ASP.NET Core MVC**, **Entity Framework Core**, **SQL Server** e boas práticas de desenvolvimento.

---

##  Tecnologias utilizadas

* ASP.NET Core MVC (.NET 8)
* Entity Framework Core
* SQL Server
* Bootstrap 5
* jQuery
* HTML5
* CSS3

---

##  Funcionalidades

###  Dashboard

* Indicadores de clientes cadastrados
* Total de ordens de serviço
* Ordens abertas
* Ordens em andamento
* Ordens finalizadas
* Ordens canceladas
* Total faturado

###  Clientes

* Cadastro
* Edição
* Exclusão
* Pesquisa por nome
* Pesquisa por CPF/CNPJ
* Validação de e-mail
* Validação de CPF/CNPJ duplicado

###  Ordens de Serviço

* Cadastro
* Edição
* Exclusão
* Filtro por status
* Controle de abertura e finalização
* Associação com clientes
* Status coloridos
